Project Overview

This is a full-stack blogging platform designed for personal use, where you (the admin) can securely publish blogs, and visitors can read your posts without any login or signup. The platform features a clean, modern UI with dark/light mode support and a mobile-friendly, responsive layout.

Key Features

  • Admin-Only Publishing: Only the admin (you) can log in and publish new blogs using a secure password-protected form. The admin interface is accessible via a dedicated button and is hidden from regular users.
  • Public Blog Viewing: All visitors can read published blogs without needing to log in or sign up. Blogs are displayed in a clean, readable format with support for long-form content.
  • Dark/Light Mode Toggle: Users can switch between dark and light themes for a comfortable reading experience, with a persistent toggle button.
  • Modern UI/UX: Custom profile and X (Twitter) icon linking to your social profile. Responsive design using Tailwind CSS. Smooth transitions and clear separation between admin and user views.
  • Inline Blog Detail View: Clicking on a blog post opens it in a focused, inline detail view, making it easy to read long posts and view images.
  • Secure Backend: Node.js/Express API with JWT-based admin authentication. MongoDB Atlas for scalable, cloud-based blog storage. RESTful endpoints for blog management.
  • Easy Deployment: The project is structured for easy deployment of both frontend and backend, with environment variables for secure configuration.

Technologies Used

React Node.js Express MongoDB Tailwind CSS JWT
