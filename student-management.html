Project Overview

The Student Management System is a comprehensive full-stack web application designed to streamline the management of student records within educational institutions. It facilitates functionalities such as adding, updating, deleting, and viewing student information through a user-friendly interface. This project was developed during GLAU@HackSpace 2.0 hackathon conducted by GLA UNIVERSITY, where our team secured the runner-up position.

Key Features

  • Student Record Management: Create, read, update, and delete (CRUD) operations for student data, including personal details and academic information.
  • Responsive Design: Ensures usability across various devices with a responsive frontend layout.
  • Modular Architecture: Structured codebase separating frontend and backend concerns for maintainability.
